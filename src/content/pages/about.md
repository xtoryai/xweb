## About This Site

This is a demo site powered by XWeb — an open-source AI-readable website engine.

XWeb is built on Astro 6 SSR + Sveltia CMS, with built-in Schema.org JSON-LD structured data output.

### Tech Stack

- **Framework**: Astro 6 (SSR)
- **CMS**: Sveltia CMS (headless, Git-based)
- **Styling**: Tailwind CSS v4
- **Runtime**: Node.js standalone
