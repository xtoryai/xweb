## Welcome to XWeb

This is a demo site powered by XWeb — an open-source website engine built on Astro 6 SSR + Sveltia CMS.

### Key Features

- **AI-Readable**: Built-in Schema.org JSON-LD structured data
- **Source Delivery**: Apache-2.0, full source code included
- **Headless CMS**: Sveltia CMS with visual Markdown editing
- **Zero Database**: Markdown files + Git version control

### Getting Started

1. Visit `/admin/` to access the CMS admin panel
2. Create posts and pages in Markdown
3. Customize the theme and layout
4. Build and deploy anywhere

Enjoy building!
